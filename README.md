Analizador Climático – ClimaPy

ClimaPy es una aplicación desarrollada en Python que permite consultar y analizar información meteorológica de cualquier ciudad. Obtiene datos climáticos en tiempo 
real de una fuente confiable en Internet para que los usuarios puedan ver la temperatura, la velocidad del viento y otros aspectos del clima.

Funciones:
Dar al usuario informacion sobre el clima en tiempo real a traves de una interfaz grafica.
Poder filtrar por ciudad a traves de la longitud y latitud.
Guardar los datos para poder analizarlos más tarde.
ver datos anteriores y tambien posteriores segun se vayan publicando.

En que se diferencia:
Crea un historial de datos de diferentes ciudades.
Genera graficos para su facil comprension.
